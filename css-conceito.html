CSS - Conceitos

CSS - Cascading Style Sheets, ou "Folhas de Estilo em Cascata", permite a estilização do conteúdo HTML. Seu foco está nos aspectos visuais (estilização, design, formatação).

Assim como a HTTML, CSS não é uma linguagem de programção, também é escrito de forma declarativa. Permite a alteração de cores, estilos de texto e posicionamento dos elementos.

Pode ser definido no código HTML de 3 maneiras:

  • Atributo style (inline)
  • Tag style (colocada no cabeçalho / head)
  • Arquivo externo (melhor , inclusive pela possibilidade de reuso)

Sintaxe

A sintaxe básica de um cósigo CSS é:

h1 (chamado de seletor) {
padding: 10px; (declação)
margin (propriedade): 0 auto;
color: blue (valor);
}

Perceba como primeiro definimos qual estrutura será editada, através do seletor. Depois abrimos um bloco usando chaves, e dentro desse bloco ficará os comandos. A linha inteira de comando é chamada de declaração. O atributo que será alterado chamamos de "propriedade", e o valor atribuído a essa propriedade, chamamos de "valor". Repare também que toda linha de comando termina com o uso de ponto-e-vírgula (;)

Quando é declarado dessa forma, toda a tag h1 que existir na página html que receber essa edição terá essas mesmas propriedades.

Para fazermos uma edição de forma satisfatória, temos que diferenciar 3 conceitos:

  • Elemento - estiliza todos os elementos conforme o identificador. Representado pelo nome da tag
  • Classe - estiliza todos os elementos que possuam o atributo classe (class). Representando no CSS por um ponto (.)
  • Id - estiliza o elemento que possua o id especificado (atributo id). Geralmente deve ser feito para um único elemento. Representado no CSS por uma hashtag (#)

Algumas Propriedades:

  • padding - margem interna
  • margin - margem externa, e também é uma das formas de centralizar o elemento
  • color - definição da cor do texto

Quando vamos alterar o tamanho da fonte, é interessante usar o padrão "rem", pois ele trabalha com um tamanho relativo e não absoluto. 1rem é o tamanho que a fonte já está, 2rem a fonte dobra de tamanho. Dessa forma o site se torna mais responsivo

CSS Reset

É uma técnica para remover o CSS padrão dos navegadores, o que auxilia na padronização visual do site. Para que funcione, deve ser feita antes de qualquer outra declaração CSS. Existem diversas ténicas para isto, a mais usada é a inserção do arquivo reset.css criada por Eric Meyer.

Os frameworks geralmente já vem com isso embutido como padrão.
