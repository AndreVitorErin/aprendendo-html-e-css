Tags HTML

  • <h1> → título principal
  • <h2> → título secundário
  • <h3> a <h6> → outros títulos seguindo a ordem de importância (quanto menor o número, maios a importância
  • <div> → divide uma sessão do navegador (divisão)
  • <img> → tag sem fechamento; usada para adição de imagens; contém parâmetros que podem ser usados
  • <header> → cabeçalho
  • <hgroup> → é um conteiner para um conjunto de elementos <h1><h6>
  • <br> → tag sem fechamento; quebra de linha
  • <wbr> → word break ou quebra de palavra; serve para hifenização do texto, ou seja, aponta que caso seja necessário alguma palavra pode ser quebrada naquela ponto. Muito útil para palavra com hífem, que caso contrário, seriam lidas como uma palavra so. Também é muito útil para url de sites muito grandes, para não comprometer a formatação, colocar a tag após cada ponto
  • <del> → efeito riscado sobre a palavra. E indica semanticamente que foi um texto deletado.Exemplo: riscado
  • <span> → utilizada para formatar pequenos techos de texto
  • <sup> → sobrescrever o texto. Exemplo sobrescrito
  • <sub> → subscreve o texto. Exemplo subscrito
  • <code> → tag específica para formatar código. Não altera a aparência, mas informa para o navegador que aquilo faz referência a um trecho código de programação.
  • <pre> → tudo que estiver entre essa tag será considerado como pré-formatado. Utilando juntamente com a tag code, é ideal para postar código. Exemplo de um código em C++

Edição CSS

O CSS possui várias opções de edicação de texto, imagem, background, etc. Não seria possível dar conta de tudo numa única página, mas vamos abordar alguns mais utilizados

text-decoration

Edita diretamente a parência do texto, pode receber os atributos:

  • underline → sublinha o texto. Exemplo
  • overline → faz um traço cima do texto. Exemplo
  • line-through → risca o texto Exemplo
  • none → gera um texto sem decoração alguma. Importante para se livrar de uma herança de característica. Exemplo

font-weight

Gerencia o negrito. Pode receber os atributos:

  • normal → texto sem formatação alguma de negrito.Exemplo
  • bold → negrito tradicional.Exemplo
  • bolder → negrito ligeramente mais intenso.Exemplo
  • definir → você pode digitar um valor entre 100 e 900 para definir a intensidade do negrito (sendo 400 a configuração padrão e 900 a intensidade máxima, e qualquer valor abaixo de 400 uma formatação ligth).Exemplo 900 e Exemplo 100

text-align

Esse parâmetro é responsável pelo alinhamento do texto. Pode receber os atributos:

  • center → centraliza o texto.
  • left → valor padrão: alinha o texto à esquerda
  • rigth → alinha o texto à direita
  • justify → justifica o texto
